# tic-tac-toe-python
<h1>Project Overview</h1>
This project is a command-line Tic-Tac-Toe game built using Python. The game allows two players to take turns placing Xs and Os on a 3x3 board. It includes input validation, win-condition checking, draw detection, and board updates after each move.

<h1>Features</h1>

- <b> Interactive command-line gameplay</b>
- <b>Two-player mode</b>
- <b>Input validation to prevent invalid moves</b>
- <b>Automatic win detection</b>
- <b>Draw detection when the board is full</b>
- <b>Clear board display after each turn</b>

<h2>Technology Used</h2>

- <b>Python 3</b>

<h2>Skills Demonstrated</h2>

- <b>Python programming fundamentals</b>
- <b>Functions and modular code design</b>
- <b>Loops and conditional statements</b>
- <b>Lists and data structures</b>
- <b>User input handling</b>
- <b>Problem-solving and game logic development</b>

<h2>Environments Used </h2>

- <b>MacOS</b>
- <b>VS Code</b>
